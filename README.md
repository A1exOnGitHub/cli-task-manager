# CLI Task Manager

A modular command-line task manager built with Python using object-oriented design and JSON persistence.

## Features

- Add tasks
- Remove tasks
- Mark tasks complete
- Persistent storage using JSON
- Auto-incrementing unique task IDs
- Modular architecture

## Technologies Used

- Python
- OOP principles
- File I/O
- JSON serialization

## How to Run

```bash
python main.py
```

## Project Structure

- task.py -> Task model
- task_manager.py -> Management logic
- storage.py -> Persistence layer
- main.py -> CLI interface
